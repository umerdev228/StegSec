About
Welcome to StegSec!
This application offers two core functionalities
Hide Data
  • 1. Select the file you wish to hide (txt, doc, doc, xisx, xls, pptx, ppt, pdf).
  • 2. Choose a cover file (Image, Audio, Video) to conceal the data.
  • 3. Choose an output directory and file name for the new file
  • 4. Enter the password you wish to use to secure the data.
  • 5. The Steg Operation and output file extension are automatically defined based on selection.
  • 6. Click the "Hide" button to begin.
Extract Data
  • 1. Select the multimedia file from which you want to extract data.
  • 2. Manually choose the operation type for extraction.
  • 3. Choose an output directory and file name for the extracted data.
  • 4. The output file extension are automatically defined based on selection.
  • 5. Enter the password you used to secure the data.
  • 6. Click the "Extract" button to begin.
Supported File Formats
  • . Cover Files: png, jpg (Images), mp3, wav (Audio), mp4, avi, mkv (Video)
  • . Files to Hide: txt (Text), doc, doc (Word), xisx, xIs (Excel), pptx, ppt (PowerPoint), pdf (PDF)
Note
  • . For PDF files, the data may get corrupted. It's advised to use multimedia files that are at least 10 times larger than the PDF file you are hiding?
  • . Additional Note, You can stop the operation at any time by clicking the "Stop" button.
